# Penetration Tester

These notes are based on the **Penetration Tester Job Role Path** from Hack The Box and are intended for you if you are starting out in information security and working towards becoming a professional penetration tester.

This path is designed to give you a solid grounding in security assessments by walking you through the full penetration testing lifecycle. As you work through it, you will build an understanding of how real-world assessments are carried out, including the tools, techniques, and structured methodologies used by practitioners in the field.

You will start with the fundamentals and gradually move into more specialised topics. Along the way, you will practise each phase of a penetration test, beginning with reconnaissance and enumeration and continuing through exploitation, post-exploitation, and finally documentation and reporting. The emphasis is not only on learning tools, but also on developing the correct mindset required to think like an attacker while maintaining a professional and methodical approach.

By the time you complete this path, you should be comfortable performing intermediate-level security assessments against enterprise-style environments. You will have both the theoretical background and hands-on experience needed to approach penetration testing engagements in a structured and repeatable way. It is strongly recommended that you are already familiar with core information security concepts, as covered in the Information Security Foundations skill path, before working through these modules.
