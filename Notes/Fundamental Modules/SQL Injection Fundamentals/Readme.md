# SQL Injection Fundamentals

SQL injection (SQLi) is one of the oldest and most impactful vulnerabilities in web security, consistently appearing in the OWASP Top 10. It occurs when user-supplied input is embedded directly into a SQL query without proper sanitisation, allowing an attacker to manipulate the query's logic.
