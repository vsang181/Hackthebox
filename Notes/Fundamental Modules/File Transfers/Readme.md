# File Transfers

This module introduces various techniques for transferring files **to and from** target Windows and Linux systems. It primarily focuses on *living-off-the-land* approaches, leveraging built-in operating system utilities to our advantage.

Operating systems, security controls, and monitoring capabilities can vary significantly across environments. As a result, a single file transfer method may not always be effective. The techniques covered in this module are designed to prepare us for a wide range of scenarios, including situations where traditional tools or external services are blocked.

Throughout the module, we explore methods that enable us to:

* Download tools or payloads onto a target system
* Retrieve files from a remote system for offline analysis on our attack box
* Adapt to restrictive network controls and host-based defenses

By understanding and applying these techniques, we can remain effective even in tightly controlled or heavily monitored environments.
