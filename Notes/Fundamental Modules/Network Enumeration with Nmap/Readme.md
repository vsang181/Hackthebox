# Network Enumeration with Nmap

Nmap (Network Mapper) is a widely used tool for discovering and enumerating systems on a network. It plays a critical role in network diagnostics, security assessments, and penetration testing by helping identify what hosts are present, what services they are running, and how they are configured.

In the context of internal network assessments, Nmap is commonly used to:

- Identify live hosts on a network segment
- Perform port scanning to discover open TCP and UDP ports
- Conduct service enumeration to determine what applications and versions are running
- Perform operating system detection based on network behaviour and responses
- Assist in attack surface mapping prior to exploitation or deeper enumeration

Throughout this module, the focus is on using Nmap efficiently and responsibly to build an accurate picture of an internal network. By combining host discovery, targeted port scans, and service detection, Nmap enables a structured and methodical approach to network enumeration, forming the foundation for later stages of assessment and analysis.
