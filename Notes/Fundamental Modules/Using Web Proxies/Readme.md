# Using Web Proxies

Burp Suite and OWASP ZAP are the two dominant web proxy tools in penetration testing, and mastering their core features gives you a strong foundation for any web application assessment. The tools covered in this module represent the everyday workflow, but both platforms go considerably deeper.
