# Cross-Site Scripting (XSS)

XSS is consistently ranked among the most prevalent web vulnerabilities. Where SQL injection targets the database tier, XSS targets the browser of anyone who visits the affected page, turning the trusted web application into a vehicle for delivering malicious JavaScript to real users.
