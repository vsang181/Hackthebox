# Login Brute Forcing

Login brute forcing is the process of systematically testing large numbers of username and password combinations against an authentication endpoint until valid credentials are found. The two primary tools covered in this module are Hydra and Medusa, both of which support a wide range of protocols and are standard in penetration testing toolkits.
