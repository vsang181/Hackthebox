# Introduction

Welcome to the **JavaScript Deobfuscation** module.

Code deobfuscation is an essential skill for anyone involved in **code analysis, reverse engineering, and security operations**. During red team and blue team exercises, we frequently encounter obfuscated code designed to conceal its true functionality. A common example is malware that uses obfuscated JavaScript to retrieve or execute its primary payload.

Without a clear understanding of what obfuscated code is doing, it becomes difficult—or impossible—to accurately assess its intent, impact, or behaviour. This lack of visibility can prevent analysts from successfully completing red team or blue team objectives.

---

## Module Overview

In this module, we begin by reviewing the **general structure of an HTML page** and learning how to locate embedded or linked JavaScript code. Once JavaScript code is identified, we introduce the concept of **code obfuscation**, including:

* What obfuscation is
* Why it is used
* Where it is commonly encountered

We then move on to practical **JavaScript deobfuscation techniques**, followed by analysing the deobfuscated code to understand its behaviour. Finally, we examine how to manually replicate functionality and uncover the true purpose of the code.

## Topics Covered

This module will cover the following topics:
