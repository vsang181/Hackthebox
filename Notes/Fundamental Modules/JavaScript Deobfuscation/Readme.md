# JavaScript Deobfuscation

Malicious actors frequently obfuscate their code to reduce the likelihood of detection by security controls and to hinder analysis by other developers or analysts.

The ability to deobfuscate code is a valuable skill with practical applications in real-world environments. During web application assessments, it can be used to identify cases where developers rely on security by obscurity to conceal JavaScript logic that may contain sensitive information. From a defensive perspective, deobfuscation is equally important, such as when analysing scripts used in phishing campaigns or other malicious web-based attacks.
