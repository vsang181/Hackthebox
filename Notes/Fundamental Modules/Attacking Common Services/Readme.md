# Attacking Common Services

Organisations rely on a common set of network services -- file sharing, remote access, email, databases, and web infrastructure -- that each carry their own attack surface. When these services are misconfigured, running outdated software, or using weak credentials, they become reliable entry points for attackers regardless of how well the rest of the environment is secured.
