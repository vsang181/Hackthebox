# Using the Metasploit Framework

**[Metasploit Framework](https://www.metasploit.com/)** is an open-source penetration testing platform developed and maintained by Rapid7 that consolidates network enumeration, vulnerability exploitation, payload delivery, post-exploitation, privilege escalation, and detection evasion into a single unified toolset. It is the most widely adopted exploitation framework in professional penetration testing and red team operations, used across every phase of an engagement from initial reconnaissance through to full post-exploitation and reporting.
