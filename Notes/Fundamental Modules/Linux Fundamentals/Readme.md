# Linux Fundamentals

Linux is an essential operating system in cyber security, and you will encounter it constantly in real-world environments. A large proportion of servers run on Linux, which makes it a critical platform for penetration testers, defensive security teams, and systems administrators alike. If you want to work effectively in this field, you must be comfortable interacting with Linux systems, especially through the terminal.

These notes focus on helping you get started with Linux in a practical way. You are not expected to become an expert immediately. The goal is to give you a solid foundation so that you can navigate the operating system confidently, understand what is happening under the hood, and interact with services and files without hesitation.

You will learn the core concepts required to work efficiently with the Linux operating system and its shell. This includes understanding how the file system is structured, how commands are executed, and how to use the terminal as your primary interface. As you progress, this knowledge will become second nature and will support almost every other topic you study in cyber security.

Treat this module as a baseline. The more time you spend practising these fundamentals, the easier it will be to adapt to new tools, environments, and attack scenarios later on.
